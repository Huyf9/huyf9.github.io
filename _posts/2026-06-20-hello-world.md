## 写作流程

1. 在 Obsidian 中打开本仓库作为库（Vault）
2. 在 `_posts` 目录下新建 Markdown 文件
3. 使用 **Obsidian Git** 插件提交并推送到 GitHub
4. GitHub Actions 自动构建，约 1–2 分钟后站点更新

## 文件命名规则

文章文件名必须为：

```text
YYYY-MM-DD-文章标题.md
```

例如：`2026-06-20-hello-world.md`

## Frontmatter 模板

每篇文章开头需要 YAML 元数据：

```yaml
---
title: 文章标题
date: 2026-06-20 12:00:00 +0800
categories: [分类名]
tags: [标签1, 标签2]
---
```
